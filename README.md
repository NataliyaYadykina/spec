##Итоговая проверочная работа

**Задача:**

Написать программу, которая из имеющегося массива строк формирует массив из строк,

длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести

с клавиатуры, либо задать на старте выполнения алгоритма. При решении не рекомендуется

пользоваться коллекциями, лучше обойтись исключительно массивами.

*Примеры:*

["hello", "2", "world", ":-)"] => ["2", ":-)"]

["1234", "1567", "-2", "computer science"] => ["-2"]

["Russia", "Denmark", "Kazan"] => []

**Решение**

1. Предложим пользователю указать, какое количество элементов он хочет ввести в массив.

2. Проверим, что введено значение больше нуля. Если введено значение 0 или меньше 0, 
сообщим пользователю об ошибке.

3. Инициализируем пустой массив с указанным пользователем количеством элементов.

4. Напишем метод, который будет предлагать пользователю ввести элементы в созданный массив. 
Тут же будем считать, какое число введенных элементов массива содержит количество символов 
меньше либо равное 3. Метод будет возвращать данное количество.

5. Инициализируем новый массив, размерность которого будет соответствовать полученному 
в пункте 4 количеству интересующих нас элементов.

6. Напишем метод, который на вход будет принимать исходный массив и новый массив. 
В новый массив запишем элементы, количество символов в которых меньше либо равно 3. 
Метод будет возвращать новый, уже заполненный, массив.

7. С помощью метода вывода массива покажем исходный и новый массив в консоли.
